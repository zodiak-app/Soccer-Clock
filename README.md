# Soccer-Clock
Soccer Clock

Du bist ein ambittioniertes Vereinsmitglied und suchst ein Tool welches dich als Stadionsprecher oder Turnierleiter unterstützt?
Soccer Clock kann viel mehr als Zeit und Spielstand erfassen, du kannst auch eine günstige Anzeigetafel (HDMI & TV) anbieten und Jingles abspielen - der Hallenmodus bringt alles mit um ein tolles Turniererlebnis zu erzeugen.
